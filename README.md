# Study Planner Application

A simple, client-side study planner app built with HTML, CSS, and JavaScript.

## Features
- Add subjects and tasks with due date and priority
- Mark tasks completed/uncompleted
- Delete individual tasks
- Clear completed or all tasks
- Today view shows tasks due today
- Data stored in `localStorage` (persisted across refresh)

## Run
1. Open `index.html` in your browser.
2. Start adding tasks.
